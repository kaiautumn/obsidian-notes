# 插入公式
点击AxMath插件，点击行内公式
撤入的情况下，双击公式即可重新编辑

在直接点开Axmath独立运行时，ctrl+s直接保存
ctrl + tab 点开Latex脚本编辑区
# 语法
### 快捷键，右键点击数学公式可以自定义快捷键
1. ctrl + 滚轮 缩放布局
2. ctrl + f  生成一个分数
3. ctrl + h  输入指数
4. ctrl + l  输入下角标
5. ctrl + r  输入根号
6. ctrl + g 英文字母变成希腊字母，并且输入一个后自动切换回英文格式。比如输入a成α，输入p成pi
或者直接反斜杠\就有了
7. 自定义符号：找到度分秒的栏目，点击小人头像
