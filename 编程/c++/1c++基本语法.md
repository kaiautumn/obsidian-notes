## 运算
1. **优先级**：`!` > 算术 > 关系 > `&&`、`||` > 赋值。
2. **求余运算**：
   - 整数才可求余，余数的符号与左边数的符号相同。
   - 示例：`3%2=1`, `-3%2=-1`, `3%-2=1`, `-3%-2=-1`, `8%4=0`。
3. **结合方向**：
   - 大多数运算符从左到右。
   - 赋值、单目运算符、条件运算符从右到左。
4. **单目运算符**：`+`, `-`, `++`, `--`, `!`, `&`, `sizeof`, `~`, `*`（解引用）。
5. **条件运算符**（三元运算符）：`?:`，优先级高于赋值运算。
6. **逗号表达式**：整个表达式的值是最后一个表达式的值，优先级最低。

## 指针
1. 指针变量未赋值前，不要用 `*` 改变取值。
2. 安全赋值方法：`int *p = new int;`。
3. 指针只能进行加减和关系运算。若 `p` 指向数组元素 `a[5]`，则 `q = p + 3` 指向 `a[8]`。
## 数组
1. 数组大小必须为常量，不能动态定义。
2. 数组名是首地址，等价于指针：
   - `a[i]`, `p[i]`, `*(a+i)`, `*(p+i)` 等价。
3. **二维数组**：
   - 初始化方式：`{{}, {}}` 或 `{,,,,}`。
   - 第一维长度可省略，第二维不可省略。
   - `a` 表示首地址，`a[0]` 表示第 0 行首地址，`&a[0][0]` 是第一个元素地址。
4. **字符串数组**：
   - 初始化方式：`char str[5] = {'x', 'j', 't', 'u'}` 或 `char stu[] = "xjtu"`。
   - 字符串以 `\0` 结尾。
## const
const是一个关键字，用于声明常量，修饰变量、参数和成员函数。
1、常量声明：
```
const int MAX=100
//一旦初始化就不能再次修改其值
```
2、常量修饰变量
```
const int x=10;
```
3、常量引用
```
const int& ref=x;
//声明了一个常量引用ref，不能通过ref修改x的值
```
4、在定义类的成员函数的时候，让这些成员函数不改变类的数据成员，也就是只读函数（有的成员函数是需要改变类数据成员的值的）。把不改变数据成员的函数后面加上const关键字进行标识，以提高代码的可读性。
const 加载前面表示返回值类型为const
加在后面表示不可修改class的成员

### 深拷贝浅拷贝
浅拷贝就是简单的复制拷贝操作，深拷贝就是在堆区重新申请空间进行拷贝工作
```
#include<iostream>
using namespace std;
class Person{
public:
	int age;
	int *height;
	Person(int a_ge,int h_eight){
		age=a_ge;
		height=new int(h_eight);
	}
	Person(const Person& p1){
		age=p1.age;
		//height=p1.height;->浅拷贝，会导致 重复删除堆区的新地址
		                     并且编译器默认是这种拷贝构造
		height=new int(*p1.height);
	}
	~Person(){
		if(height!=NULL){
			delete height;
			height=NULL;
		}
	}
};
int main(){
	Person p1(18,160);
	Person p2(p1);
}
```
