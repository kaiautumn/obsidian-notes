# 基本语法  
1. 段落：空行来分割  多个空格和空行会被合并  
2. 小标题：#的数量
3. 粗体：**粗体文本**        %%ctrl + B%%
4. 斜体：*斜体文本*        %%ctrl + I %%
5. 删除线：~~删除线文本~~
6. 高亮：==高亮==
7. 注释：\**不会被加粗\**
8. 内部链接：\[[标题]]  \[标题]\(标题.后缀)
9. 外部链接：\[描述链接的文本]\(URL)             %%ctrl + k%%
10. 引用：文本前加>符号来引用文本,如：
>这是一个引用

11. 列表：-或 * 或 +，后空格

- 第一条
- 第二条

12. 任务列表：-[]，记得加空格         %%直接对选中文字ctrl+l%%
- [x] 已完成
- [ ] 未完成

13. 水平线：三个连续的---

---

14. 代码：三个反引号
```
include<iostream>
using namespace std;
int main(){
}
```
15. 表格 ，表格上下空出一行即可，或者直接右键
| 表头 | 表头 |
| --- | --- |
| 单元格 | 单元格 |
| 单元格 | 单元格 |

16. 脚注：[ ^1 ] 
17. 注释：四个%包裹
%%你说的%%
注释内容只会在编辑视图中可见。

# 快捷键
ctrl+B选中文字加粗
ctrl+/选中文字注释
ctrl+l选中文字便任务列表
ctrl+x剪切
ctrl+z撤销   ctrl+y恢复
ctrl+g打开关系图谱
ctrl+ delete/backspace 删除前一个或者后一个的单词
ctr +shift+k删除当前行(未选择文本的时候)
# 编辑与预览笔记
ctrl+E更换视图
按住ctrl的时候点击切换，可以同时打开同一个笔记的两个视图
# 标签
“#”+主题词
搜索时，在搜索框 tag:#标签
嵌套标签 : 在标签名中加/
# 标注
在第一个位置加>符号
类型：[!note]笔记
	   [!abstract]摘要
	   [!info]信息
	   [!todo]代办
	   [!tip]注意
	   [!success]已做
	   [!question]问题
	   [!warning]警告
	   [!failure]失败
	   [!danger]危险
	   [!bug]bug
	   [!example]例子
	   [!quote]引用
实例：
>[!note] 笔记
>[!note]后加一个空格，紧跟标题，下一行是内容

标注折叠：
>[!bug]+
>在[!bug]后加一个+号或-，可以默认展开或收起


嵌套标注：可以多层标注
>[!note]
>>[!example]
>>> [!note]甚至多层嵌套
